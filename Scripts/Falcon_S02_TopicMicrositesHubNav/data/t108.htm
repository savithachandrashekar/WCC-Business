Find the most updated information for Windows 10 update related issues along with tips on actions to take before installing the update on your PC.

Before you update

Before you update the Windows 10 versions of your computer, here are a few recommended steps that should be completed

 

  1. Check product compatibility:

    Installing Windows 10 updates on non-compatible products may result in issues that may break functions or features. HP keeps a record of all products that are tested for newer Windows 10 versions. For tested products, HP will provide updated driver and software. Click here to view a list of tested products

  2. Run Performance tune-up:

    HP performance tune up can help improve system performance and free-up space so that updates can download and install on you PC.

  1. Update BIOS:

    BIOS is a key element that ensures newer updates and existing components can work together. Ensure that you have the latest BIOS installed in your PC before updating Windows 10. To learn more about how to install updated BIOS, visit the documents below.
    • Updating the BIOS for HP Laptops
    • Updating the BIOS for Desktops

  2. Update Graphics Drivers:

    An out-of-date graphics driver can cause the Windows update to fail and roll back. It is recommended to check and ensure that the latest Graphics drivers are installed in your computer before starting the update process. Visit the Software Download Page to select your product and download the latest driver.

  3. Check for updates:

    Select the Start button, and then go to Settings > Update & security > Windows Update, and select Check for updates. If Windows Update says your device is up to date, you have all the updates that are currently available. For additional information visit Updating drivers and software with Windows update.

How to use Bluetooth on Windows 10 HP PCs

Bluetooth wireless technology provides short-range communications that replace the physical cable connections that traditionally link electronic devices, such as printers, audio devices, and computers. Some examples include:

  • Sharing pictures with other Bluetooth devices nearby
  • Sending files to a Bluetooth printer
  • Transferring files to nearby Bluetooth computers
  • Listening to music on a Bluetooth headset

How to tune-up PC and improve system performance?

There are several reasons that could be impacting your computer and degrading performance.

  1. Computer Might Be Infected by a Virus or Malware
    This document explains viruses and malware, provides instructions for removing viruses and malware, and offers suggestions for preventing future infections.
    • Computer Might Be Infected by a Virus or Malware
  2. Computer is Slow
    Chat with our HP Virtual agent for instructions on how to speed up the computer and improve slow system performance.

  1. Low Disk Space/ Disk Clean up
    Windows uses disk space for caching files and virtual memory. If the free space on the hard drive drops too low, files do not save, discs do not burn, and the computer or tablet might become unreliable.
    Find out what you can do to increase disk space and enhance performance
    • Resolving Low Disk Space Errors in Windows 10
  2. Update BIOS
    Updating the BIOS, also known as "flashing the BIOS," replaces the BIOS firmware. When completed successfully, a BIOS update can fix or enhance aspects of a computer's performance, or provide support for newly installed hardware.
