How to solve microphone problems

There are many possible causes of microphone problems, and these basic steps should help resolve the most common ones. If you're using an external microphone with an app such as Zoom or Mesenger the problem could be due to the headset or the configuration of the app.

This video contains the basic steps to fix the most common microphone issues.

Resolving Microphone Problems (Windows 10, 8) contains detailed steps for solving microphone issues.

If you have an external microphone and still have problems, we recommend visiting the manufacturer's support website.

If you're using an app such as Zoom or Skype, you may need to check its audio configuration. Use the app's help, or go to the relevant support website for more information. For example:

  • Zoom support

  • Skype support

  • Messenger support

The HP Support Communities may have information, or you can ask the experts directly.
