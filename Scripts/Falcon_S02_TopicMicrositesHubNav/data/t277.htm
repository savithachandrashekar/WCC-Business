How to use automated tools to resolve sound issues

You have one of these symptoms: no sound coming from the speakers, no volume or the audio is not working, sound is muffled, intermittent, crackling/popping or distorted.

There are many reasons why sound doesn’t work. The automated solutions we recommend here should fix most Windows 10 problems. If you have a different version of Windows, try the links the the end of the section for help.

If you have issues after updating Windows 10, see the section I'm having issues after updating Windows 10 on this page.

Two automated tools are available:

  • HP Audio Check is the recommended tool. It detects common hardware problems and also checks the HP driver is working correctly. It requires HP Support Assistant 8.5 or later.

  • The Windows Audio Troubleshooter doesn't require HP Support Assistant, but doesn't test the HP drivers.

Troubleshooting with HP Audio Check

  1. Restart your computer.

  2. Run HP Audio Check and follow the instructions.

If the Audio Check says it is working correctly, any remaining problems are probably not due to the HP computer hardware or software.

Windows Audio Troubleshooter

If HPSA isn't installed or you don't want to install it, use the Windows Audio Troubleshooter.

  1. Restart your computer.

  2. In Windows, type find and fix audio into Search, or right-click the speaker icon on the taskbar and select Troubleshoot sound problems.

  3. Select Find and fix audio playback problems or Find and fix audio recording problems, and click Next.

  4. Select the device you want to troubleshoot, and then click Next and follow the instructions.

  5. If the Windows Audio Troubleshooter says it is working correctly, any remaining problems are probably not due to the HP computer hardware or Windows software.

How to test sound if the audio checks don't find problems

If you are using external speakers or headphones, they might be faulty. Try using another set of speakers, or a pair of headphones. If the second set of speakers or headphones produces sound without problems, the first set is probably faulty.
